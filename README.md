# Enterprise Dairy Management Platform

An enterprise-grade Android application ecosystem developed in Java for automating end-to-end dairy operations through separate **Admin** and **Super Admin** applications. The platform enables dairy businesses to efficiently manage farmers, milk collection, billing, inventory, rate charts, analytics, and system administration using a scalable Firebase Firestore backend.

## Key Features

* Dual Android applications (Admin & Super Admin)
* Farmer Management System
* Real-time Milk Collection
* Dynamic Rate Chart Management
* Automatic Bill Generation
* Inventory Management
* Reports & Analytics Dashboard
* Audit Log Tracking
* Multi-tenant Firebase Firestore Architecture
* Material Design 3 UI with modern dashboards

## Technologies Used

* Java
* Android SDK
* Firebase Firestore
* Firebase Authentication
* MVVM Architecture
* Material Design 3
* RecyclerView
* ViewModel & LiveData
* Git & GitHub
